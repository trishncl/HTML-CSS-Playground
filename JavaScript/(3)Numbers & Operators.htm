Arithmetic Operator

Operators that can perform mathematical equations on numbers
Operator Function Example
+ Addition 5 + 5
- Substraction 5 - 5
= Assignment let num = 12
* Multiplication 5 * 5
/ Division 5 / 5
** Exponent 5 ** 5
% Modulus (Remainder) 5 % 2
Operators can be applied to variables as well.

Arithmetic Precedence

PEMDAS is followed in the order of precedence.

  1. Parentheses
  2. Exponent
  3. Multiplication
  4. Division
  5. Addition
  6. Subtraction

Shorthand Operators

Performs an arithmetic operator to a variable and assigns it to itself.
Operator Usage Same as
+= x += 5 x = x + 5
-= x -= 5 x = x - 5
*= x *= 5 x = x * 5
/= x /= 5 x = x / 5
**= x **= 5 x = x ** 5
%= x %= 5 x = x % 5
