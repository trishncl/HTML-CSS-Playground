Variables

Temporary container that can hold different types of data such as text, numbers, collections, objects, and etc. Variables can be named for easy read/write access by the programmer it is called an identifier.

Identifiers

variable name that has been set by the programmer.

  1. Identifiers must be UNIQUE
  2. Identifiers are CASE SENSITIVE
  3. RESERVE KEYWORDS are NOT allowed as Identifiers
  4. Identifiers MUST start with a LETTER, $ or _
  5. Identifiers CANNOT contain SPECIAL CHARACTERS

4 Ways of Declaring Variables

  1. Automatically
  2. Using var
  3. Using let
  4. Using const

Automatically

Automatic variables automatically declare themselves.

*x is an example of identifier

Using let

Using the let keyword, it cannot be redeclared. It is preferred to be used by default when declaring variables. (Recommended way)

Using const

Uses the const keyword, it cannot be redeclared these variables cannot be reassigned/changed.

Using var

Uses the var keyword, it can be redeclared and should only be used when you want to support older browsers.
