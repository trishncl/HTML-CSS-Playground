CSS

CSS (Cascading Style Sheet) is the standard language used to style an HTML Document.

CSS determines how to display the HTML Document in a more appealing way. Adding colors, formatting etc.

*Currently using CSS3 which is the current standard and still updated up to this day.*

CSS Syntax

How is it structured?

                        p{
                            color: red;
                            font-size: 24px;
                        }

<(style)> Tag

Used to integrate a Stylesheet (CSS) to our HTML Document without creating an external CSS File.
