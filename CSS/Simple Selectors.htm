Selectors

Used to find the HTML elements that we want to style or design using CSS. It can select any number of HTML elements and apply a certain style for them.

                (head)
                    (style)
                        p{
                            color: red;
                            font-size: 50px;
                        }
                    (/style)
                (/head)
                (body)
                        (p)Hello World!(/p)
                (/body)

Simple Selectors

Selects element/s by using either their:
  • Tag Name - Name of the HTML Element
  • Class Name - General Name assigned to Elements
  • ID - Unique Name assigned to an Element
Used before indicating the selector:
  • Tag Name (only the tag)
  • Class Name - use dot before the selector (.p)
  • ID - used hashtag before the selector (#p)
                div{
                    color: white;
                }

                .class{
                    background: orange;
                }

                #id{
                    background: yellow;
                }
            *ID is more specific than the class so the system will read the ID.*
