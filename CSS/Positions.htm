Positions

CSS Positions

Indicates the position behavior of an element.

                    p{
                        position:static;
                    }

                    nav{
                        position:sticky;
                        top:0;
                    }
Value Description
static Positioned according to the flow of the page.
fixed Positioned on the same place even if scrolled.
sticky Sticks in positioned place when not in viewport.
absolute Positioned relative to its positioned parent.
relative Positioned relative to its defaul position.

CSS Positioning Properties

Indicates the margin edges of a positioned element.

*This does not work with static positioned elements.
Value Description
top Top edge margin for a positioned element.
right Right edge margin for a positioned element.
bottom Bottom edge margin for a positioned element.
left Left edge margin for a positioned element. 
                    .action-button{
                        position:fixed;
                        bottom:10px;
                        right:10px;
                    }
