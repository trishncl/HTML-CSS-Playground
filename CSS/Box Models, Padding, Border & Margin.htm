Box Models, Padding, Border & Margin

CSS Box Model

  • Content - Anything inside an Element can be considered a content.
  • Padding - Invisible spacing between the content and the border.
  • Border - Black lines
  • Margin - Invisible spacing outside the border.

Padding

It is an invisible space between the content and the border.

                    .container-1{
                        padding-top:10px;
                        padding-right:10px;
                        padding-bottom:10px;
                        padding-left:10px;
                    }

                    *padding:y-padding x-padding;*
                    .container-2{
                        padding:10px 25px;
                    }

                    *padding:top right bottom left;*
                    .container-3{
                        padding:5px 10px 5px 10px;
                    }

Border

A visible line that separates the content and other elements.

                    *border:width style color;*
                    .container-1{
                        .border:2px solid tomato;
                    }
Value Description
dotted Dotted Lines
dashed Broken Lines
solid Solid Line
double Double Lines
groove 3D Grooved Border
ridge 3D Ridged Border
inset 3D Debossed Border
outset 3D Embossed Border

Border Controls

Borders can be styled differently on each side.

                        .container-1{
                            border-top:2px solid tomato;
                            border-right:3px dotted tomato;
                            border-bottom:4px dashed tomato;
                            border-left:5px double tomato;
                        }

Border Smoothness

Borders can become rounded to achieve a modern feeling.

                        .container-1{
                            border-top:2px solid tomato;
                            border-radius:20px;
                        }

Margin

It is an invisible space between the border and other elements.

                    .container-1{
                        margin-top:10px;
                        margin-right:10px;
                        margin-bottom:10px;
                        margin-left:10px;
                    }

                    *margin:y-margin x-margin;*
                    .container-2{
                        margin:10px 25px;
                    }

                    *margin:top right bottom left;*
                    .container-3{
                        margin:5px 10px 5px 10px;
                    }
