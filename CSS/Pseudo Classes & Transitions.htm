Pseudo Classes & Transitions

CSS Pseudo Classes

Adds style to special states of an element.

Such as hover, click, disabled and etc.

Common Pseudo Classes
Selector Description
:hover Mouse Over State
:active Clicked / Active Links
:focus Focused on fields
:hover Mouse Over State
:checked Selected Radioboxes / Checkboxes
:disabled Disabled Inputs/Buttons
Full Pseudo Classes List

We can use pseudo classes by adding it after a selector.

                        button{
                            background-color:tomato;
                        }

                        button:hover{
                            background-color:red;
                        }

                        button:active{
                            background-color:blue;
                        }

CSS Transitions

Adds a smooth transition in between style changes.

                button{
                    background-color:tomato;
                    transition:1s;
                }

                 button:hover{
                    background-color:red;
                }

                button:active{
                    background-color:blue;
                }

Provides specific properties for smooth transition. 

                *transition:property duration*
                button{
                    background-color:tomato;
                    transition:scale/background-color/opacity/width/height 2s;
                }

                *transition:duration delay*
                button{
                    background-color:tomato;
                    transition:1s 2s;
                }

                   Timing-Function
                *transition:timing duration*
                button{
                    background-color:tomato;
                    transition:linear 2s;
                }

                button:hover;
                    background-color:red;
                    scale:1.1;

                *Shorthand Technique*
                *transition:property duration timing delay;*
                button{
                    background-color:tomato;
                    transition:scale 1s linear 2s;
                }
