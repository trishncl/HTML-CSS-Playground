Text Formatting

Text Appearance
Property Description
color Changes the color of the text
text-transform Changes the letter of the text
text-shadow Adds a shadow effect on the text 

        3 Values of text-transform:

            h1{
                text-transform: uppercase;
            }

            h2{
                text-transform: lowercase;
            }

            h3{
                text-transform: capitalize;
            }

        Text-Shadow

            Syntax:
                selector{
                    text-shadow:h-pos v-pos blur color;
                }

                p{
                    text-shadow:1px 1px 10px red;
                }

Fonts

It is the overall appearance of a digital & printed text; it has a certain style and can be changed accordingly.

Property Description
font-size Changes the size of the text
font-family Changes the the font of the letters
font-style Changes the font style of the words
font-weight Changes the thickness of the font
font-variant Indicates a variant of a font
