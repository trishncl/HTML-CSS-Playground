Backgrounds

Background Color

Colors can be set using:

  • Color Names (pre-defined)
  • RGB (Red, Green, Blue)
  • RGBA (Red, Green, Blue, Alpha)
  • HSL (Hue, Saturation, Lightness)
  • HSLA (Hue, Saturation, Lightness, Alpha)
  • HEX
            body{
                background-color: tomato;
            }

Background Images

            body{
                background-image:url('bg.png')
            }

Background Repeat

  • repeat
  • repeat-x
  • repeat-y
  • round
  • space
  • no-repeat
                .container-1{
                    background-image:url('bg.png')
                    background-repeat: repeat;
                }

                .container-2{
                    background-image:url('bg.png')
                    background-repeat:repeat-x;
                }               
                
                .container-3{
                    background-image:url('bg.png')
                    background-repeat:repeat-y;
                }

                .container-4{
                    background-image:url('bg.png')
                    background-repeat:round;
                }

                .container-5{
                    background-image:url('bg.png')
                    background-repeat:space;
                }

                .container-5{
                    background-image:url('bg.png')
                    background-repeat:no-repeat;
                }

Background Attachment

  • scroll
  • fixed
                    .container-1{
                        background-image:url('bg.png')
                        background-attachment: scroll;
                    }
    
                    .container-2{
                        background-image:url('bg.png')
                        background-attachment:fixed;
                    }

Background Position

Syntax

background-position: x-pos y-pos;

background-position:right/center/left top/center/bottom

                    .container-1{
                        background-image:url('bg.png')
                        background-position:right top;
                    }

                    .container-2{
                        background-image:url('bg.png')
                        background-position:50% 50%;
                    }

                    *0% - left most side | 100% - right most side*

                    *0% - top most side | 100% - bottom most part

Background Size

  • auto
  • contain
  • cover
                    .container-1{
                        background-image:url('bg.png')
                        background-size:auto
                    }

                    .container-2{
                        background-image:url('bg.png')
                        background-size:contain
                    }

                    .container-2{
                        background-image:url('bg.png')
                        background-size:cover
                    }

                    container-2{
                        background-image:url('bg.png')
                        background-size: 400px 200px;
                    }

                    *Syntax 
                    background-size: x-size y-size;

Background Shorthand

                body{
                    background:url('bg.png') fixed no-repeat center top / cover;
                }

                *body{
                    background:url('bg.png') bg-attachment bg-repeat bg-position / bg-size;
                }*
