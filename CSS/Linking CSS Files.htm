Linking CSS Files

Ways to link CSS to HTML File

  1. Internal CSS
  2. External CSS
  3. Inline CSS

Internal CSS

CSS that uses the (style) tag inside the HTML Document.

                (p style="font-size: 13px; color: bisque")(i)*Currently using (b)CSS3(/b) which is the current standard and still updated up to this day.*(/i)(/p)

External CSS

Requires a separate CSS File and makes use of the (link) tag for us to be able to link the CSS File to our HTML Document.

                (head)
                    (link rel="stylesheet" href="style.css")
                (/head)

Inline CSS

This is the fastest way to integrate styles to our HTML Document.

This can also make your code VERY HARD to read so DON'T use it unless you really need to.

It uses the style attribute in all the elements.

                (p style='color:red; font-size:100px')Hello World(/p)
