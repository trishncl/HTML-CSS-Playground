Colors & Units

CSS Colors

Colors can be set using:

  • Color Names (pre-defined)
  • RGB (Red, Green, Blue)
  • RGBA (Red, Green, Blue, Alpha)
  • HSL (Hue, Saturation, Lightness)
  • HSLA (Hue, Saturation, Lightness, Alpha)
  • HEX
We have 140 pre-defined colors supported 
                Pre-defined Colors
                    
                    body{
                        background: greenyellow;
                    }

                    p{
                        color: tomato;
                    }
                
                RGB Colors
                    Syntax - rgb(red, green, blue)
                    Values 0 - 225
                        
                        body{
                            background:rgb(50, 255, 100)
                        }
                    
                        p{
                            color:rgb(255, 255, 255);
                        }
                    
                RGBA Colors
                    Syntax - rgba(red, green, blue, alpha)
                    Values 0 - 225 (RGB) | 0-1 (Alpha)
                        
                        body{
                            background:rgba()
                        }

                        p{
                            color:rgba(255, 255, 255, 0.5)
                        }

                HSL Colors
                    Syntax - hsl(hue, saturation, lightness)
                    Values 0 - 360 (hue) | 0% - 100% (sat. & lig.)

                        body{
                            background: hsl(270, 100%, 50%)
                        }

                        p{
                            color: hsl(0, 100%, 50%)
                        }

                        

                        

                        

                HSLA Colors
                    Syntax - hsl(hue, saturation, lightness, alpha)
                    Values 0 - 360 (hue) | 0% - 100% (sat. & lig.) | 0-1 (alpha)
                        
                        body{
                            background:hsla(270, 100%, 50%, 0.7)
                        }

                        p{
                            color:hsla(0, 100%, 50%, 0.3)
                        }
                
                HEX
                Syntax#RRGGBB

                        body{
                            background: #000000;
                        }

                        p{
                            color: #ffffff
                        }

CSS Units

These are used in sizes on CSS. Such as the 'font-size', 'width', and 'height'.

2 categories of Units
  • Absolute Lengths
  • Relative Lengths
Absolute Lengths
Unit Description Conversion
cm centimeters
mm millimeters
in inches 1in = 96px = 2.54cm
px pixels 1px = 0.0104in
pt points 1pt = 0.01389in
pc picas 1pc = 12 pt

Relative Lengths
Unit Description
em Relative to the font-size of the element.
ex Relative to the x-height of the current font.
ch Relative to width of the character "0".
rem Relative to font-size of the root element.
vw Relative to 1% of the width of the viewport*.
vh Relative to 1% of the height of the viewport*.
vmin Relative to 1% of viewport's* smaller dimension.
vmax Relative to 1% of viewport's* larger dimension.
% Relative to the parent element.
