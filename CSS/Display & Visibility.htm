CSS Display

Indicates the display behavior of an element.

                    *both of this attribute has its value as their defaults.
                    p{
                        display:block;
                    }

                    span{
                        display:inline;
                    }
Common CSS Displays
Value Description
block Takes up full width and starts in a new line.
none Removes element from layout
inline Takes up content width (not affected by Width & Height)
inline-block Takes up content width (affected by Width & Height)
flex Flex Container
grid Grid Container
