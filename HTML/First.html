Best Practices in HTML

  1. Use Proper Document Structure
  2. Always Specify the !DOCTYPE HTML
  3. USE LOWERCASE Markup Tags
  4. Always Close Tags
  5. USE Proper Indentation
  6. Use ONLY 1 (h1) Tag per Page
  7. USE Semantic Elements where possiblle
  8. Label Non-Semantic Elements meaningfully
  9. USE (div) or Semantic Tags to dovode the sections of your website
  10. Always use meaningful meta tags
