HTML Entities

Special Characters that are usually not accessible by the keyboard we instead use codes to display them in our HTML Document, it is also used to display reserved keywords from html like the "<" and ">" symbol.


HTML Entities are displayed by using an ampersand "&" followed by an "entity name", "entity number" or an "entity hexadecimal" and eding with a semicolon ";"

HTML Emojis

It looks like an icon but is actually a character, Emojis are characters from the UTF-8 charset, Emojis are also entities so it is accessed as same as the other HTML Entities, although they don't have an "Entity Name"


Using UTF-8 Encoding: (meta charset="UTF-8"/)

HTML Charsets

ASCII

It is the first character encoding standard. ASCII had 128 different characters that could be used: numbers (0-9), English letters (A-Z), and some special characters like !$+-()@<>.

ISO-8859-1

It is the default character encoding of HTML 4 this character set had 256 characters.

ANSI

It was the original Windows character encoding it is same as the ISO-8859-1 but this has 32 more characters.

UTF-8 (Unicode)

Is the modern standard if character encoding it supports 1,112,064 charactersthis includes the symbols and emojis.
