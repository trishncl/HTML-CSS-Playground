Source Element (source>)

indicates files for the browser to choose from.
source tags are used within (

Example: (source src="music.mp3" type="audio/mp3">)
            (source src="music.ogg" type="audio/ogg">)

Source Element Attributes

src - indicates the file path, can take a value of a URL (direct link) or a file path
type - indicates the MIME-type of the resource
MIME-type helps identifying file types so it can be hadled accordingly
Mime Type Lists (https://www.freeformatter.com/mime-types-list.html)

Audio Element (audio>)

used to play audio files.

                
                or

Audio Element Attributes

src - indicates the file path
controls - shows controls like play, pause
autoplay - play audio automatically when loaded (doesn't work on some browsers)
muted - audio would be muted by default
loop - makes the audio replay infinite times
src attribute - (audio src="audio.mp3"/)
other attributes (doesn't need any value) - (audio src="audio.mp3" controls autoplay loop muted/)

Video Element (video>)

use to play video files.
            
            or

Video Element Attributes

same with audio element attributes, only added with width and height
src attribute - (vid src="vid.mp4" width="320" height="240"/)
other attributes - (vid src="vid.mp4" width="320" height="240" controls autoplay loop muted/)
