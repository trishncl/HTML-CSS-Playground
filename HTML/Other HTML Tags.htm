Other HTML Tags

<(meta)> Tag

Defines metadata about the HTML file, it should always be placed inside the (head) tag.

The (meta) tag does not display anything in the webpage but is essential in giving instructions on how the webpage is displayed or searched in search engines.

                (html)
                    (head)
                        (meta charset="UTF-8")
                        (meta name="description" content="Free Programming Tutorials")
                        (meta name="keywords" content="Programming, HTML, BSIT")
                        (meta name="author" content="ncl")
                        (meta name="viewport" content="width=device-width, initial-scale=1.0")
                    (head)
                (html)

                These (3) values are the most important in <(meta)> tag: description, keywords, author.
                Practice including <(meta)> tag in any website you're doing for it can help the Search Engine Optimization (SEO).

<(link)> Tag

Defines a "link" between the HTML file to an external source file, such as favicons and stylesheets (css).

It is normally used in the (head) tag to associate an icon or a css file to the html document.

                    (html)
                        (head)
                            (link rel="icon" href="favicon.ico"/)
                            (link rel="stylesheet" href="style.css"/)
                        (/head)
                    (/html)

<(script)> Tag

It is used to include a client-side script which is JavaScript.

It can be used by either including a JavaScript code inside the tag or using an external file using the src attribute.

                    (script)
                        alert('Hello World!')
                    (/script)

                    (script src='somescript.js')(/script)

<(iframe)> Tag

It is udes to embed an HTML Document within an HTML Document.

It is often used to include buttons or content from other websites.

                    (html)
                        (body)
                            (iframe src="https://biyaya-starosa.web.app" title="Biyaya Application"/)
                        (/body)
                    (/html)

<(progress)> Tag

It is used to diplay a progress bar. It is commonly used to display tasks progress.

(label) tags can also be used for the progress tags for accessibility.

                    (html)
                        (body)
                            (progress value ="75" max="100"/)
                        (/body)
                    (/html)
                    
                    We can manipulate it using the JavaScript.
