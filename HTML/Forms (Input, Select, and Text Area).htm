Form Element

  • indicates a form
  • form tags are used in combination with other elements like (input), (textarea), and (select)
    •                     Example: (form action="/action_page.php" target="_blank")
                                (input type="email"/)(br/)
                                (input type="password"/)(br/)
                                (input type="submit"/)
                             (/form)


    (form) Element Attributes

  • action - specifies the action to perform after submitting the form.
  • method - specifies the type of method that the form will use "GET" or "POST".
  • target - specifies where the result will display after submitting the form.

    • Input Element

    indicates a textfield, radiobutton, checkbox etc.

    (input) tag has a lot of variety, we can change this by using the "type" attribute.

                            Example: 
                            (input type="text"/)
                            (input type="password"/)
                            (input type="radio"/)
                            (input type="checkbox"/)
                            (input type="range"/)
                            (input type="number"/)
                            (input type="date"/)
                            (input type="file"/)

    (input) Element Attributes

    These attributes are used generally for all input types some attributes are based on the input type.

  • type - specifies the type input.
  • name - specifies the name of the input (passed to server).
  • value - specifies the default value of the input.
  • required - specifies that the field is required.
  • disabled - specifies that the field is disabled.
  • readonly - speifies that the field can only be read.

    • (input) Element Text Field Types

  • text - a normal text field.
  • email - a text field with an email address validation.
  • password - a text field that hides the text in it.
  • number - a text field that only allows numbers.
  • tel - a text field that has a telephone number format validation.
  • search - a text field for searching.
  • url - a text field with url validation.

    • (input type="text") Attributes

  • placeholder - placeholder text shown in the textfield.
  • minlength - minimum number of characters allowed.
  • maxlength - maximum number of characters allowed.
  • size - size of the textfield.

    (input) Element Date/Time Types

  • date - a field for inputting the date/month/year
  • datetime-local - a field for inputting the date/month/year and time
  • month - a field for inputting the month and year
  • week - a field for inputting the week and year
  • time - a field for inputting thetime
    • default format: YYYY/MM/DD
    default format (time): hh:mm

    (input) Element Date/Time Attributes

  • min - sets the minimum date/time the user can pick
  • sets the maximum date/time the user can pick

    • (input) Element Button Types

  • file - a button that allows selection of files from local storage.
  • checkbox - a toggleable button, allows multiple selection
  • radio - a toggleable button, allows only one selection
  • button - a classic button
  • reset - a button that resets all the value of the form
  • submit - a button for submitting the form

    • (input type="file") Attributes

  • accept - specifies the files accepted
  • multiple - specifies if multiple files are accepted
  • capture - specifies which camera to use for capture of image or video data


    • (input type="checkbox") Attributes

  • checked - specifies if the checkbox is checked by default.


    • (input type="radio") Attributes

  • checked - specifies if the radio button is selected by default.



    • *To group an input element, put a "name" attribute. The value of name should be the same for you to group it.





    (input) Element Other Types

  • range - a field that lets you set a value using a slider.
  • color - a field for selecting color.
  • hidden - a field that is not visible.

    • (input type="range") Attributes

  • step - interval of steps to take when changing the value of the slider.
  • min - minimum value of the slider
  • max - maximum value of the slider




    • (textarea) Element

    indicates a text area

                                Example: (textarea rows="4" cols="50")
                                        This is a sample text
                                    (/textarea)

    (select) Element

  • indicates a drop-down lsit
  • the (select) element is used in combination with the (option) element.
    • Example:
                            (select name="members")
                                (option value="Trisha")Trisha(/option)
                                (option value="Nicole")Trisha(/option)
                                (option value="Labay")Trisha(/option)
                                (option value="Sañosa")Trisha(/option)
                            (/select)

    (select) Element Attributes

  • multiple - specifies if the user can select multiple items in the list.
  • size - specifies number of options visible on the drop-down.
  • name - specifies the name of the textarea
  • required - specifies that the field is required
  • disabled - specifies that the field is disabled
  • form - specifies the form it belongs to using the form's id

    • (option) Element

    indicates an item in a drop-down list

    (optgroup) Element

    indicates an item group in a drop-down list

    Example:
                                (select name="College")
                                  (optgroup label="BatStateU")
                                    (option value="IT 111")IT 111(/option)
                                    (option value="CS 111")CS 111(/option)
                                  (/optgroup)
                                  (optgroup label="students")
                                    (option value="Trisha")Trisha(/option)
                                    (option value="Nicole")Trisha(/option)
                                    (option value="Labay")Trisha(/option)
                                    (option value="Sañosa")Trisha(/option)
                                  (/optgroup)
                                (/select)

    (optgroup) Element Attributes

  • label - specifies the name of the item group
  • disabled - specifies that the item group is disabled

    • (label) Element

    indicates a label for an (input), (textarea), and (select) element


    the (label) element "for" attribute uses an id value to indicate what element they are bound to.

                                    (label for="txtFullName")Full Name:(/label)
                                (input type="text" id="txtFullName" name="fullName")

                                (br)

                                (label for="slcSex")Sex:(/label)
                                (select id="slcSex")
                                    (option value="M")Male(/option)
                                    (option value="F")Female(/option)
                                (/select)
